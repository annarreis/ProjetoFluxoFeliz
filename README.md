# 🚀 Projeto Fluxo Feliz com Git e GitHub

## 📌 Descrição
Este projeto foi desenvolvido com o objetivo de praticar o fluxo de versionamento utilizando Git e GitHub, simulando um ambiente real de desenvolvimento.

Durante o desenvolvimento, foram aplicados conceitos como criação de branches, commits descritivos, abertura de Pull Requests e merge de código.

---

## ⚙️ Funcionalidades
- Estrutura básica de uma página web
- Estilização com CSS
- Interação com JavaScript
- Exibição dinâmica de mensagem ao clicar em botão

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Git
- GitHub

---

## 🔄 Fluxo de desenvolvimento

O projeto seguiu um fluxo baseado em boas práticas de versionamento:

1. Criação de branches para cada melhoria
2. Desenvolvimento isolado das funcionalidades
3. Commits descritivos
4. Abertura de Pull Requests
5. Revisão e merge na branch principal (main)

---

## 🌿 Estrutura de branches

- `main`: versão principal do projeto
- `feature/interatividade`: adição de conteúdo e melhoria estrutural
- `feature/adiciona seção sobre o projeto e remove linha duplicada`: correção de erros e adição de seção sobre no index.
